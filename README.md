# Luxe 3D Showcase

A premium Flutter application designed to showcase products in an interactive 3D environment. This app verifies the implementation of 3D model visualization, clean architecture, and modern UI design principles.

## Features

- **3D Product Interaction**: Users can rotate, zoom, and interact with 3D product models using `model_viewer_plus`.
- **Modern UI/UX**: Features a dark maroon and white theme with glassmorphism effects, smooth animations, and a clean layout.
- **State Management**: Utilizes the Provider package for efficient state management across screens.
- **Product Details**: Displays dynamic product information including price, rating, colors, and quantity selection.

## Technology Stack

- **Flutter**: UI Toolkit for building natively compiled applications.
- **Dart**: Programming language used for logic.
- **Provider**: For state management.
- **Model Viewer Plus**: For rendering interactive 3D GLB models.
- **Google Fonts**: utilized Poppins for typography.
